# Gemini Diffusion

A text-to-text diffusion model using [tinygrad](https://github.com/tinygrad/tinygrad).

## Setup

```bash
uv venv .venv
source .venv/bin/activate
uv pip install --upgrade pip
uv pip install -r requirements.txt  # or use pyproject.toml with uv
```
